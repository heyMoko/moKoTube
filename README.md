# moKoTube
📽 motionLayout과 exoplayer, retrofit을 이용한 영상플레이어 📽

## 개발환경

* OS : windows 10 pro
* IDE : Android Studio
* Language : Kotlin

## 사용한 것
* MotionLayout
* ExoPlayer
* Retrofit
